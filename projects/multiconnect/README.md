# Multiconnect Part Generators

Multiconnect-based mounting and storage solutions using BJD's Multiconnect standard.

[View on MakerWorld](https://makerworld.com/en/models/582260)

## Structure

- `holders/` - Various item holders (vertical, round single, round row)
- `shelves/` - Shelf modules
- `storage/` - Storage bins
- `hooks/` - Hook accessories
- `adapters/` - Adapters for other systems (Gridfinity, rolled steel)
- `in-progress/` - Work in progress designs
