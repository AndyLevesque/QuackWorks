# Underware - Infinite Cable Management

Underware is a modular cable management system using parametric OpenSCAD channels and accessories.

[View on MakerWorld](https://makerworld.com/en/models/783010)

## Structure

- `channels/` - Various channel types (I, C, L, T, X, Y, S, Diagonal, Transition, Branch Split, Height Change, Mitre)
- `accessories/` - Cable loop holders, item holders, hooks, connectors
- `scripts/` - Python build scripts for batch generation
- `in-progress/` - Work in progress designs
- `deprecated/` - Deprecated files (kept for reference)

## Build Script

Use `scripts/underware-builder.py` with `scripts/configuration.yaml` for batch generation.
