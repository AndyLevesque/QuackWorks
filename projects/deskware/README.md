# Deskware - Modular Desk System

Deskware is a modular desk organization system.

[View on MakerWorld](https://makerworld.com/en/models/1331760-deskware-a-modular-desk-system)

## Files

- `deskware-main.scad` - Main deskware module
- `screw-mount.scad` - Screw mount accessory
- `hok-connector.scad` - HOK connector module
- `exports/` - Exported STL/3MF files
